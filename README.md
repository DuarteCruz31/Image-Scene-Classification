# Image Scene Classification of Landscapes
```bash
conda install -c conda-forge opencv

```

## Dataset
- [Intel Image Classification](https://www.kaggle.com/puneet6060/intel-image-classification)

## Models
- CNN
- VGG16
- ResNet50
- DenseNet121

## Results
- CNN: 0.85
- VGG16: 0.88
- ResNet50: 0.60
- DenseNet121: 0.89
