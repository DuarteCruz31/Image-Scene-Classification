# Image Scene Classification of Landscapes

## Dataset
- [Intel Image Classification](https://www.kaggle.com/puneet6060/intel-image-classification)

## Models
- CNN
- VGG16
- ResNet50
- DenseNet121
- KNN

## Results
- CNN: 0.85
- VGG16: 0.88
- ResNet50: 0.60
- DenseNet121: 0.90
- KNN: 0.37
